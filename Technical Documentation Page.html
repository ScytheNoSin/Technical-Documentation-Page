Introduction

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It tells the browser how to structure and display content. HTML elements are the building blocks of all websites.

HTML is not a programming language — it’s a markup language. You use it to annotate text, images, and other content to tell the browser what they are and how they should be displayed.

Here is a simple example of an HTML document:

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, world!</h1>
    <p>Welcome to my first web page.</p>
</body>
</html>
Getting Started

To start with HTML, you need a basic understanding of its syntax and structure. HTML uses "elements" that are wrapped in angle brackets. Most elements have an opening tag, content, and a closing tag.

For example, a paragraph tag looks like this:

<p>This is a paragraph.</p>

Some tags are self-closing, meaning they don’t need a closing tag. Example:

<img src="image.jpg" alt="My Image" />

HTML elements can also have attributes, which provide extra information:

<a href="https://example.com">Visit Example</a>
Examples

HTML provides a wide range of elements for text, images, links, lists, and more.

  • <h1> - <h6>: Headings
  • <p>: Paragraphs
  • <img>: Images
  • <ul> / <ol>: Lists
  • <div> and <span>: Generic containers

Using these elements correctly helps browsers and assistive technologies render your content properly.

Conclusion

Semantic HTML introduces elements that clearly describe their meaning. It makes your code easier to read and improves accessibility.
